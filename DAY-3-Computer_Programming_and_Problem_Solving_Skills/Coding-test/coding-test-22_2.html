ให้รับข้อมูลผ่านทาง Input คะแนนของผู้เรียนเข้ามาทางคีย์บอร์ด
หากได้คะแนนตั้งแต่ 90 คะแนนขึ้นไป จะได้เกรด 4 หากได้คะแนน 80 - 89 คะแนน จะได้เกรด 3
หากได้คะแนน 70 - 79 คะแนน จะได้เกรด 2
หากได้คะแนน 60 - 69 คะแนน จะได้เกรด 1
หากได้คะแนนต่ำกว่า 60 คะแนน จะได้เกรด 0
หากรับข้อมูลคะแนนมามากกว่า 100 คะแนน หรือ น้อยกว่า 0 ให้แสดงข้อความว่า "Error"
